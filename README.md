# react 组件库搭建

[在线预览地址](https://minjiechang.github.io/mj-design)
## 技术栈

1. 开发打包

typescript + gulp + babel + inquirer

2. 文档解析

docz + plop

## 启动项目

```shell
git clone https://github.com/MinjieChang/mj-design.git

cd mj-design

yarn

yarn start
```

## 新建组建

```shell
yarn run new
```

## 发布文档

```shell
yarn run deploy:doc
```

## 发布组件到npm

```shell
yarn run release
```
